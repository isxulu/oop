## Compile

You can compile my program using the following command:

+ In windows:

  ```c++
  g++ -std=c++11 -o main.exe .\main.cpp
  ```

  + and run by:

    ```c++
    .\main.exe
    ```

+ In linux:

  ```c++
  g++ -std=c++11 -o main ./main.cpp
  ```

  + and run by:

    ```c++
    ./main
    ```
